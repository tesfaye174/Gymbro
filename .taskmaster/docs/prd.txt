# GymBro - PRD (Product Requirements Document)

## 1. Executive Summary

App pensata come strumento da allenamento, non come app generica fitness.
Flussi ridotti a 1 tocco, timer automatici e modalità Stand ottimizzata per uso senza mani.
Architettura offline-first con logging affidabile delle sessioni e progressione chiara.

## 3. Flussi Utente Principali

### 3.1 Allenamento del Giorno

Flow: 1 tap → start → timer parte → auto-advance.

Wireframe:
```
[Header]
Oggi - Push Day

[Tempo totale stimato]
45 min

[Lista sintetica esercizi]
Bench Press - 4x8
Shoulder Press - 3x10

[CTA grande]
INIZIA ALLENAMENTO
```

Micro-interazioni:
- Tap start → haptic leggero
- Timer parte subito
- Schermo rimane attivo

### 3.2 Creazione Scheda

```
Nome scheda
Tipo split
+ Aggiungi esercizio

[Lista esercizi]
> Drag per riordinare
```

Interazioni:
- Template rapidi
- Duplicazione con swipe

### 3.3 Durante Allenamento

```
Nome esercizio
Serie 2/4

[TIMER GRANDE]
00:32

Peso attuale
[ + ] [ − ]

Avanti automatico
```

Zero scroll.

### 3.4 Modalità Stand

Layout centrato:
```
TIMER GIGANTE
Nome esercizio

Serie
Peso

[ Pulsanti grandi in basso ]
PAUSA   SALTA
```

### 3.5 Visuale Progressi

- Grafico peso nel tempo
- Record personale
- Streak giorni

## 5. Timer e Logica

### State Machine
```
IDLE → EXERCISE_RUNNING → REST_RUNNING → NEXT_SET → NEXT_EXERCISE → COMPLETED
```

### Comportamenti
- Vibrazione breve → fine esercizio
- Vibrazione doppia → fine recupero
- Auto-advance attivo

### Background
- Local notification programmata
- Timer interno preciso

## 9. Architettura Tecnica

- Pattern: MVVM + Combine
- Storage: Core Data
- Timer: DispatchSourceTimer
- Speech: Speech Framework
- Notifiche: UNUserNotificationCenter
- Backup: iCloud opzionale

## 10. Snippet SwiftUI

### Creazione Sessione
```swift
let session = WorkoutSession(context: viewContext)
session.id = UUID()
session.date = Date()
try viewContext.save()
```

### Timer Preciso
```swift
timer = DispatchSource.makeTimerSource()
timer?.schedule(deadline: .now(), repeating: 1)
timer?.setEventHandler {
    DispatchQueue.main.async {
        self.seconds -= 1
    }
}
timer?.resume()
```

### Scrittura HKWorkout
```swift
let workout = HKWorkout(activityType: .traditionalStrengthTraining,
                        start: startDate,
                        end: endDate)
healthStore.save(workout) { _, _ in }
```
