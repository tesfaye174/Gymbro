# Status: pending
# Dependencies: 2

# Description: Build the screen for creating and editing workout templates with exercise list, drag-to-reorder, and quick templates.
# Details:
1. Create TemplateListView showing all saved templates.
2. Create TemplateEditorView with:
   - TextField for template name
   - Picker for split type (Push, Pull, Legs, Upper, Lower, Full Body, Custom)
   - List of exercises with drag-to-reorder (.onMove modifier)
   - Each exercise row: name, sets x reps, rest time
   - Swipe to delete exercise
   - '+ Add Exercise' button
3. Create ExerciseEditorSheet:
   - Exercise name (with search/autocomplete from common exercises)
   - Stepper for sets (1-10)
   - Stepper for reps (1-50)
   - Stepper for rest seconds (15-300, step 15)
4. Create TemplateEditorViewModel handling CRUD via WorkoutRepository.
5. Support template duplication (swipe action on template list).
6. Provide 3-4 quick templates (PPL, Upper/Lower, Full Body) that can be loaded.
7. Follow dark mode design: black background, high contrast text, CTAs at bottom.

# Test Strategy:
Test creating a template with multiple exercises. Verify reordering persists. Test editing an existing template. Test duplicate creates independent copy. Verify quick templates load correctly.
