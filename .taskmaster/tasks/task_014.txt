# Task ID: 14
# Title: HealthKit Integration
# Status: pending
# Dependencies: 7, 9
# Priority: medium
# Description: Integrate with HealthKit to write workout sessions as strength training workouts and read previous workout data.
# Details:
1. Create HealthKitService:
   - Request authorization for HKWorkoutType and HKQuantityType (activeEnergyBurned)
   - Write method: saveWorkout(session:setRecords:)
   - Read method: fetchRecentWorkouts(limit:) for showing external workouts
2. Writing HKWorkout:
   - activityType: .traditionalStrengthTraining
   - start/end dates from session
   - totalEnergyBurned: estimate from duration and intensity
   - metadata: include exercise names, total sets, total weight lifted
3. Prompt for HealthKit permission on first workout completion (opt-in).
4. Settings toggle to enable/disable HealthKit sync.
5. Show HealthKit status in settings (connected/not connected).
6. Handle authorization denied gracefully.
7. Add HealthKit capability in Xcode project and Info.plist usage descriptions.

# Test Strategy:
Test on real device (HealthKit not available in simulator). Verify workout appears in Apple Health app. Test authorization flow (grant and deny). Test with HealthKit disabled.
