# Task ID: 15
# Title: Voice Commands (Speech Framework)
# Status: pending
# Dependencies: 6, 12
# Priority: medium
# Description: Implement local voice control using Speech Framework for hands-free workout control with commands: pausa, riprendi, avanti, indietro, ripeti.
# Details:
1. Create VoiceCommandService using Speech Framework:
   - SFSpeechRecognizer with locale: it-IT (Italian)
   - Continuous recognition during active workout
   - Local processing (on-device, no network required)
2. Supported commands mapping:
   - 'pausa' / 'stop' → pause timer
   - 'riprendi' / 'vai' → resume timer
   - 'avanti' / 'prossimo' → skip to next set/exercise
   - 'indietro' → go back to previous
   - 'ripeti' → repeat current set
3. Request permissions:
   - Microphone (NSMicrophoneUsageDescription)
   - Speech recognition (NSSpeechRecognitionUsageDescription)
4. Visual feedback: microphone icon indicating listening state.
5. Voice commands toggle in settings (off by default).
6. Fallback: all voice commands have corresponding physical buttons.
7. Handle noisy gym environment: require keyword prefix or activation gesture.
8. Stop recognition when workout ends or app backgrounds.

# Test Strategy:
Test each voice command triggers correct action. Test in noisy environment simulation. Test with permissions denied. Verify fallback buttons always work. Test recognition stops on workout end.
