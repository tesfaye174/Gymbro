# Task ID: 1
# Title: Xcode Project Setup & Core Data Schema
# Status: pending
# Dependencies: None
# Priority: high
# Description: Create the Xcode project with SwiftUI lifecycle, configure Core Data stack, and define the entity schema for WorkoutTemplate, Exercise, WorkoutSession, and SetRecord.
# Details:
1. Create a new Xcode project named 'GymBro' with SwiftUI App lifecycle.
2. Enable Core Data in the project.
3. Define Core Data entities:
   - WorkoutTemplate: id (UUID), name (String), createdAt (Date), exercises (relationship to Exercise, ordered, cascade delete)
   - Exercise: id (UUID), name (String), sets (Int16), reps (Int16), restSeconds (Int16), order (Int16), template (relationship to WorkoutTemplate)
   - WorkoutSession: id (UUID), date (Date), duration (Int32), completed (Bool), templateName (String), template (optional relationship to WorkoutTemplate)
   - SetRecord: id (UUID), weight (Double), repsDone (Int16), fatigue (Int16), setIndex (Int16), exerciseName (String), session (relationship to WorkoutSession)
4. Create PersistenceController with shared instance and preview support.
5. Set up MVVM folder structure: Models/, ViewModels/, Views/, Services/, Utilities/.
6. Configure Info.plist for HealthKit and microphone usage descriptions.

# Test Strategy:
Verify project builds and runs on simulator. Verify Core Data stack initializes without errors. Create and fetch a test entity to validate schema.
